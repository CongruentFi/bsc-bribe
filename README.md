# Congruent Bribe Data Generator

Congruent will airdrop bribe to `MetaM` holders who has `sGaas`, `wsGaas`, `Gaas` on Ethereum.

## Usage

To generate merkle tree for claiming:

```shell
$ yarn
$ yarn ts-node src/gen-tree.ts
```
